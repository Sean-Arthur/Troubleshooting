This repository demonstrates troubleshooting skills. It includes:

## Scripts

1. **Log Parser (`log_parser.py`)**
   - Parses device logs to identify warnings and errors.

2. **API Tester (`api_tester.py`)**
   - Checks API connectivity, latency, and response data.

3. **Network Diagnostic (`network_check.sh`)**
   - Performs ping, DNS lookup, and port 443 checks.
